1. Json

什么是JSON
JSON是一种取代XML的数据结构，和xml相比，它更小巧但描述能力却不差,由于它的小巧所以网络传输数据将减少更多流量从而加快速度。

JSON就是一串字符串 只不过元素会使用特定的符号标注。

{} 双括号表示对象

[] 中括号表示数组

“” 双引号内是属性或值

: 冒号表示后者是前者的值(这个值可以是字符串、数字、也可以是另一个数组或对象)

所以 {“name”: “Michael”} 可以理解为是一个包含name为Michael的对象

而[{“name”: “Michael”},{“name”: “Jerry”}]就表示包含两个对象的数组

当然了，你也可以使用{“name”:[“Michael”,”Jerry”]}来简化上面一部,这是一个拥有一个name数组的对象

下面是一个json的实例：

{
“company”: “google”,
“people”: [
{
“firstName”: “Brett”,
“lastName”: “McLaughlin”,
“age”: 100
},
{
“firstName”: “Jason”,
“lastName”: “Hunter”,
“age”: 80
}
]
}

Jsoncpp

Jsoncpp是一个使用C++语言实现的面向对象的json库。

特点

  1. 读写JSON文件。
  2. 读JSON文件时，可以将C++风格的注释读取到JSON元素中。
  3. 重写JSON文件，保留原始注释。

注意：注释以前在JSON中受支持，但由于可移植性而被删除（Python不支持类似C的注释）。由于注释在配置/输入文件中很有用，因此保留了此功能。

Jsoncpp提供的接口中有3个核心类：Reader、Writer、Value。
Reader类负责从字符串或者输入流中加载JSON文档，并进行解析，生成代表JSON文档的Value对象。
Writer类负责将内存中的Value对象转换成JSON文档，输出到文件或者是字符串中。
Value类的对象代表一个JSON值，既可以代表一个文档，也可以代表文档中一个值。如同JSON中定义的”值”一样，Value是递归的。

在线安装

# centos
yum install jsoncpp
yum install jsoncpp-devel
# ubuntu
apt-get install libjsoncpp-dev

解析对象和数组

#include <iostream>
#include <jsoncpp/json/value.h>
#include <string>
#include <jsoncpp/json/json.h>
using namespace std;


int main()
{

    string str_array = "{\"people\":[{\"firstName\":\"Brett\",\"lastName\":\"McLaughlin\",\"age\":100},{\"firstName\":\"Jason\",\"lastName\":\"Hunter\",\"age\":80}]}";
    string str = "{\"firstName\":\"Brett\",\"lastName\":\"McLaughlin\"}";


    Json::Reader reader;
    Json::Value root;
    if (reader.parse(str_array, root))
    {
        string json_pp = root["people"].toStyledString();
        
        // 打印people字段信息
        cout << "打印people字段信息" << endl;
        cout << json_pp << endl;
        
        
        Json::Reader reader_pp;
        Json::Value root_pp;
        if (reader_pp.parse(json_pp, root_pp))
        {
            cout << "打印people各个字段信息" << endl;
            cout << root_pp.toStyledString() << endl;
            for(int i = 0; i < root_pp.size();i++)
            {
                Json::Value obj = root_pp[i];
                
                cout << "第" << i << "字段信息是："<< endl;
                cout << "firstName" << obj["firstName"].asString()<< endl;
                cout << "lastName" << obj["lastName"].asString()<< endl;
                cout << "age" << obj["age"].asInt()<< endl;
            }
            
        }

    }

    {
        Json::Reader reader;
        Json::Value root;
        if (reader.parse(str, root))  // reader将Json字符串解析到root，root将包含Json里所有子元素
        {

            cout << "移除对象前"<< endl;
            cout << root.toStyledString() << endl;

            if(root.isMember("lastName"))
            {
                root.removeMember("lastName");
                cout << "移除对象后"<< endl;
                cout << root.toStyledString() << endl;

            }
            
            cout << "添加对象数组前"<< endl;
            Json::Value toproot(Json::arrayValue);
            toproot.append(root);
            Json::Value subroot;
            subroot["firstName"] = "zhangsan";
            subroot["lastName"] = "lisi";
            
            toproot.append(subroot);
            cout << toproot.toStyledString() << endl;
    
        }

    }
}
