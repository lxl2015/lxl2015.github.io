制作Centos局域网本地源

  1. 1、安装http服务
  2. 2、安装相关制作的软件包
  3. 3、获取源文件
    1. 3.1、下载163的源
    2. 3.2、挂在光盘镜像
  4. 4、创建源索引文件
  5. 5、配置repo
  6. 6、验证

本来不打算记下来，网上一搜一大把，只是还要筛选一下，浪费时间



1、安装http服务

# yum install httpd

修改配置文件

# vi /etc/httpd/conf/httpd.conf

修改

ServerName www.example.com:80 
DocumentRoot "/var/www/html/yum"

重启服务

# service httpd restart

2、安装相关制作的软件包

#rpm –qa|grep  createrepo

如果为发现，则安装createrepo

#rpm –ivh createrepo-0.9.9-22.el6.noarch.rpm deltarpm-3.5-0.5.20090913git.el6.x86_64.rpm python-deltarpm-3.5-0.5.20090913git.el6.x86_64.rpm

3、获取源文件

首先先要将所有的数据包上传或者挂载到/yum/source目录下.

3.1、下载163的源

# cd /yum/source
# wget http://mirrors.163.com/centos/6/os/x86_64/Packages/
# wget -nc -B http://mirrors.163.com/centos/6/os/x86_64/Packages/ -F -nH --cut-dirs=3 -i index.html

慢慢下载吧

# ln –s /yum/source/ /var/www/html/yum/

3.2、挂在光盘镜像

对于光盘DVD，可这样挂载

# mkdir /yum/source
# mount /dev/cdrom /yum/source 
# ls /yum/source

对于iso文件，可以这样挂载

# mount -o loop rhel-server-xxx-dvd.iso /yum/source
# ls /yum/source
# ln –s /yum/source/Package/ /var/www/html/yum/

4、创建源索引文件

如果是挂载方式

# createrepo -g /media/iso/repodata/repomd.xml/ var/www/html/yum/ .

下载163的源

# createrepo /yum/source/

5、配置repo

# cp CentOS-Base.repo CentOS-Base.repo.bak
# vi CentOS-Base.repo

修改为

name=CentOS-$releasever - Base
baseurl=http://IP/
gpgcheck=0
gpgkey=http://IP/RPM-GPG-KEY-CentOS-6

6、验证

# yum clean all
# yum makecache
# yum install cmake
