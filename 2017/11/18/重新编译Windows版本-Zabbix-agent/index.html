2017-11-18

重新编译Windows版本 Zabbix agent

  1. 1、 安装环境
  2. 2、 X86编译
  3. 3、 X64编译

本来不打算记录下来，杂事太多，很快就忘记了。气愤。

1、 安装环境

* Windows SDK for Windows 7
* Visual C++ 2010 Express
* Zabbix Source
* Vmware Windows 7

所需要的SDK和Express下载好

2、 X86编译

所需软件默认安装路径

在CMD.EXE运行

“C:\Program Files (x86)\Microsoft Visual Studio 9.0\VC\bin\vcvars32.bat”

进入zabbix/build/win32/project

nmake

3、 X64编译

修改Makefile*中的文件，将

# TARGETDIR = ..\..\..\bin\win64
# ADD_LFLAGS = /MACHINE:X64

去掉注释，将

TARGETDIR = ..\..\..\bin\win32
ADD_LFLAGS = /MACHINE:X86

注释掉

在CMD.EXE运行

“C:\Program Files (x86)\Microsoft Visual Studio 9.0\VC\bin\vcvars64.bat”

进入zabbix/build/win32/project

nmake

后记：
可能因为使用的事虚拟机的原因，所以才会有修改Makefile，用物理机可能就不会有这样的问题。
