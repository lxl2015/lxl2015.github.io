Java-Web开发笔记01-eclipse环境配置

  1. 1、安装JDK
  2. 2、下载Eclipse
  3. 3、下载MYSQL 5.6
  4. 4、下载tomcat
  5. 5、Eclipse配置Tomcat

学习环境：

  • windows 10
  • Eclipse-jee-mars-2-win32-x86_64
  • MYSQL 5.6
  • Tomcat 7.0
  • JDK 1.7

1、安装JDK

下载JDK，安装即可。配置环境变量

新建

JAVA_HOME
D:\Program Files\Java\jdk1.7.0_79

JRE_HOME
D:\Program Files\Java\jdk1.7.0_79\jre

CLASSPATH
%JAVA_HOME%\lib;%JRE_HOME%\lib



在path里面添加 
%JAVA_HOME%\bin
%JRE_HOME%\bin

2、下载Eclipse

eclipse-jee-mars-2-win32-x86_64.zip

点击下面连接下载即可，绿色安装。随性配置。

3、下载MYSQL 5.6

下载完MYSQL，需配置环境变量
例如我下载的mysql解压到D:\Program Files\并重命名为MYSQL。

在MYSQL复制一份my-defaul.ini为my.ini
添加

[mysqld] 
basedir=D:\Program Files\MySQL
datadir=D:\Program Files\MySQL\data

在path里面添加

D:\Program Files\MYSQL\bin

即可。

管理员运行CMD.exe，运行如下命令

> cd D:\Program Files\MySQL\bin
> mysqld -install
> net start mysql
> mysql -u root -p #默认情况下无密码

4、下载tomcat

下载 apache-tomcat-7******即可。startup.bat启动，shutdown.bat关闭。

5、Eclipse配置Tomcat

Window——>Preferences——>Server——>Runtime Environments——>add——>后面按照提示配就可以了。

如果出现问题，Google一下。
