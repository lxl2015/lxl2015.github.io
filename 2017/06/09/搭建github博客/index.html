2、配置github

2.1、生成SSH Key

# git config --global user.name lxl2015
# git config --global user.email 1010******@qq.com

创建本地ssh(用于上传到你对应的github账号)
# ssh-keygen -t rsa -C “1010******@qq.com”

打开id_rsa.pub，将内容粘贴到github里面Settings ——>SSH and GPG kyes ——>New SSH Key ——>Add SSH Key

验证 

# ssh -T git@github.com

根据提示选择后出现

Hi Fly-Sunshine-J! You've successfully authenticated, but GitHub does not provide shell access.

说明验证成功

2.2、安装node.js和git

默认选项安装即可

2.3、安装hexo及配置

hexo官网里面更新最新的安装方式，点这里

2.3.1、安装

配置初始环境

# npm install hexo-cli -g
# hexo init blog
# cd blog
# npm install
# hexo server

hexo配置在blog下面。登录到localhost:4000预览看一下。如果界面正常，说明已经成功一半。

# vi _config.yml

按照下面内容修改

# Site
title: Linux笔记
subtitle:
description:
author: lxl2015
language: zh-CN
timezone: Asia/Shanghai 
...
...
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git
repository: git@github.com:lxl2015/lxl2015.github.io.git
branch: master

注意：你应该在github上有一个类似lxl2015.github.io的仓库

2.4、更换主题

假设你喜欢的主题叫abc

# git clone url  xxx/blog/themes/abc
# rm xxx/blog/themes/abc/.git

打开blog目录下的_config.yml

vi _config.yml

修改 

theme: abc

2.5、hexo常用命令

# hexo new "postName" #新建文章,默认在source/_posts/下面
# hexo new page "pageName" #新建页面
# hexo generate #生成静态页面至public目录
# hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
# hexo deploy #部署到GitHub
# hexo clean #清除缓存
# hexo help  # 查看帮助
# hexo version  #查看Hexo的版本

一般写文章的流程是

  1. hexo new “name” //创建文章
  2. hexo g //生成静态页面
  3. hexo s //本地预览
  4. hexo d //发布到github上面

初始化分类目录

# hexo new page "categories"

如果生成文章没有标签

# hexo new page tags

如果没有其他页面，查看主题下面的_config.yml对应的页面。

例如

about: /resume
# hexo new page resume

3、多台电脑同步HEXO博客

3.1、创建分支

在lxl2015.github.io.git创建一个分支hexo，用来存放blog文件夹下的文件。

1.删除主题下面的.git文件夹

2.在blog目录下创建hexo分支

# git init
//创建hexo分支
# git checkout -b hexo
# git remote add origin git@github.com:lxl2015/lxl2015.github.io.git
# git add package.json source _config.yml scaffolds themes
# git commit -m "add hexo"
# git push -u origin hexo

登录github，选择默认分支

3.2、另外一台电脑配置

切换分支

# git clone -b hexo git@github.com:lxl2015/lxl2015.github.io.git
# npm install

错误信息：

如果出现：ERROR Deployer not found: git

npm install hexo-deployer-git --save

出现warning: LF will be replaced by CRLF 

git config --global core.autocrlf false
