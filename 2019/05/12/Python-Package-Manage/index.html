Python包管理工具pip

pip管理工具分为两种，一种是pip2和pip3，安装的时候不要安装错了。

首先yum工具是基于python2，默认python版本是python2，当运行

yum install python-pip

默认安装的是python2-pip，所以当安装了python3版本的时候，需要运行下面命令

yum install python3-pip
