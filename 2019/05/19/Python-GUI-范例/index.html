Tkinter 教程

在这节教程中，我们将学习通过Tkinter包编写一个Python GUI范例。原文地址

Tkinter是一个Python官方标准包，我们不需要单独去安装它。它是一个非常强大的一个包。你可能使用过Python集成的开发工具IDLE，IDLE使用Tkinter编写的。
教程使用的是Python3.X。本教程你需要知道一些Python基础。本教程通过创建一个窗口来添加一些小工具，例如按钮，多选框等，然后设置它们的属性。

1. 创建第一个GUI应用

引入Tkinter包，创建一个窗口并设置标题

from tkinter import *

window = Tk()

window.title("Welcome to Python App")

window.mainloop()

运行结果如下图:

最后一行mainloop是一个无限循环的功能，等待与用户进行交互，直到窗口被关闭。

如果不调用mainloop函数，将不会出现窗口界面。

2. 创建标签控件

  • Label 标签控件；可以显示文本和位图

在前面的例子里面添加一个Label，使用label类创建一个label
lbl = Label(window, text="Hello")。

然后我们使用grid方法来设置label在窗体中的位置lbl.grid(column=0, row=0)

完整的代码如下：

from tkinter import *

window = Tk()

window.title(“Welcome to Python App”)

lbl = Label(window, text=”Hello World”)

lbl.grid(column=0, row=0)

window.mainloop()

运行结果如下：

创建label的时候不调用grid方法，label不会出现在窗体中。

7.1 获取单选按钮的值

可以设置一个变量，来获取当前选中的单选按钮的值。
完整代码:

from tkinter import *

from tkinter.ttk import *

window = Tk()

window.title(“Welcome to Python App”)

selected = IntVar()

rad1 = Radiobutton(window,text=’First’, value=1, variable=selected)

rad2 = Radiobutton(window,text=’Second’, value=2, variable=selected)

rad3 = Radiobutton(window,text=’Third’, value=3, variable=selected)

def clicked():

  print(selected.get())

btn = Button(window, text=”Click Me”, command=clicked)

rad1.grid(column=0, row=0)

rad2.grid(column=1, row=0)

rad3.grid(column=2, row=0)

btn.grid(column=3, row=0)

window.mainloop()

11. 添加进度条控件

使用progressbar类创建进度条。

from tkinter.ttk import Progressbar

bar = Progressbar(window, length=200)

也可以设置进度条的值bar['value'] = 70

首先先创建一个style，设置背景，最后设置进度条属性。

from tkinter import *

from tkinter.ttk import Progressbar

from tkinter import ttk

window = Tk()

window.title(“Welcome to Python App”)

window.geometry(‘400x200’)

style = ttk.Style()

style.theme_use(‘default’)

style.configure(“black.Horizontal.TProgressbar”, background=’black’)

bar = Progressbar(window, length=200, style=’black.Horizontal.TProgressbar’)

bar[‘value’] = 70

bar.grid(column=0, row=0)

window.mainloop()

12. 添加文件对话框（选择文件或者路径）

使用文件filedialog类创建文件对话框。

from tkinter import filedialog

from os import path

file = filedialog.askopenfilename(initialdir= path.dirname(file))

13. 创建菜单栏

使用menu类创建菜单栏。

from tkinter import Menu

menu = Menu(window)

menu.add_command(label=’File’)

window.config(menu=menu)

首先创建菜单栏，添加第一个标签，最后将菜单栏指向创建的窗口。
使用add_cascade()方法添加菜单栏的子标签。
menu.add_cascade(label='File', menu=new_item)

完整代码：

from tkinter import *

from tkinter import Menu

window = Tk()

window.title(“Welcome to Python App”)

menu = Menu(window)

new_item = Menu(menu)

new_item.add_command(label=’New’)

menu.add_cascade(label=’File’, menu=new_item)

window.config(menu=menu)

window.mainloop()

使用这个方法可以创建很多子菜单。

完整代码:

from tkinter import *

from tkinter import Menu

window = Tk()

window.title(“Welcome to Python App”)

menu = Menu(window)

new_item = Menu(menu)

new_item.add_command(label=’New’)

new_item.add_separator()

new_item.add_command(label=’Edit’)

menu.add_cascade(label=’File’, menu=new_item)

window.config(menu=menu)

window.mainloop()

可以设置tearoff属性来关闭菜单属性。
new_item = Menu(menu, tearoff=0)

添加点击事件。
new_item.add_command(label='New', command=clicked)

Add a Notebook widget (tab control)

To create a tab control, there are 3 steps to do so.

  • First, we create a tab control using Notebook class

  • Create a tab using Frame class.

  • Add that tab to the tab control.

  • Pack the tab control so it becomes visible in the window.

    from tkinter import *

    from tkinter import ttk

    window = Tk()

    window.title(“Welcome to Python App”)

    tab_control = ttk.Notebook(window)

    tab1 = ttk.Frame(tab_control)

    tab_control.add(tab1, text=’First’)

    tab_control.pack(expand=1, fill=’both’)

    window.mainloop()

Add widgets to Notebooks

After creating tabs, you can put widgets inside these tabs by assigning the parent property to the desired tab.

from tkinter import *

from tkinter import ttk

window = Tk()

window.title(“Welcome to Python App”)

tab_control = ttk.Notebook(window)

tab1 = ttk.Frame(tab_control)

tab2 = ttk.Frame(tab_control)

tab_control.add(tab1, text=’First’)

tab_control.add(tab2, text=’Second’)

lbl1 = Label(tab1, text= ‘label1’)

lbl1.grid(column=0, row=0)

lbl2 = Label(tab2, text= ‘label2’)

lbl2.grid(column=0, row=0)

tab_control.pack(expand=1, fill=’both’)

window.mainloop()

Add spacing for widgets (padding)

You can add padding for your controls to make it looks well organized using padx and pady properties.

Just pass padx and pady to any widget and give them a value.

lbl1 = Label(tab1, text= 'label1', padx=5, pady=5)
