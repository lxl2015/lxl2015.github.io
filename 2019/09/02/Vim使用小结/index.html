Vim使用小结

  1. 1. 显示十六进制文件
  2. 2. 打开比较大文件
  3. 3. vim 卡死问题
  4. 4. Root 用户无法使用vi

隐藏正文

1. 显示十六进制文件

$ vim -b file_name        # -b 参数表示以二进制打开
# 在vim命令模式输入

:%!xxd -g 1

2. 打开比较大文件

当打开比较大的文件的时候会报Out of memory saving lines for undo.

可以使用如下命令

$ vi -y 9999999 file_name

3. vim 卡死问题

有时候在使用vim的时候。会出现卡死状态，其实是使用了快捷键锁定了屏幕。

Ctrl + S 锁定屏幕
Ctrl + Q 解锁屏幕

4. Root 用户无法使用vi

当在root用户下无法使用vi命令的时候，可能是vi没有链接到vim

# vim .bashrc

添加alias vi='vim'
保存退出，执行 source .bashrc,使配置生效。
